# J-Genom (JGEM) Whitepaper v0.2

## 概要

J-Genomは、文字と読みの「乖離」という言語現象を素材とするコンセプチュアル・アートプロジェクトである。

現代のデジタル空間では文字列は一意に読まれることを前提として設計されている。しかしながら、人類の言語には文字と読みが直感的に対応しない文字列が多数存在し、そのような文字列は一意的な解釈を超えた豊かな情報密度を内包している。

J-Genomはこの「乖離」を定量化し、ブロックチェーンという恒久的な媒体に刻むことで、言語・解釈・価値の境界を問い続ける作品として自己定義する。

付記として、J-Genomの公式読みは「じゅげむ」と定める。本プロジェクト名自体がすでに乖離の一例である。

---

## 動機：なぜ乖離に価値があるか

エントロピーは「変換できない何か」を定量化するために生まれた概念である。19世紀、蒸気機関の効率を最大化しようとする工学的要請から、カルノーとクラウジウスは熱機関の理論的限界を定式化した。その過程で「熱として散逸し、仕事に変換できない何か」の定量化としてエントロピーが生まれ、やがてボルツマンによって分子の乱雑さへ、シャノンによって情報の不確かさへと接続し、自然界に遍在する不可逆性の観測へと至った。

J-Genomはこれと同じ構造を持つ。「読めない文字列」という日常的な現象を定量化しようとする試みが、言語・情報・価値の本質へと接続することを目指している。

名前のない概念は思考できない。J-Genomは「乖離」に名前と価値を与えることで、新たな思考の対象を生み出す。

### 自己帰結：乖離が乖離の上に乗る

ブロックチェーンにおけるマイニングは「計算は大変、検証は一瞬」という非対称性を価値の源泉とする。nonceを探す総当たり計算は膨大なコストを要するが、答えの検証はハッシュを一度計算するだけで完了する。この**到達コストの乖離**がProof of Workの本質であり、ネットワークの安全性の根拠となっている。

J-Genomの乖離文字列も同じ構造を持つ。読みを知らない者にとって導出コストは高いが、知っている者にとって検証は一瞬である。

すなわちJ-Genomは：
- **「計算と検証の乖離」を利用する仕組み**（ブロックチェーン）の上に
- **「文字と読みの乖離」を記録する**

という二重の乖離構造を持つ。これは設計以前に、乖離という概念の本質から自然に導かれた自己帰結である。

## 背景：乖離という現象

### 乖離の類型

文字と読みの乖離には複数の類型が存在する。

| 類型 | 例 | 特徴 |
|------|-----|------|
| 難読漢字 | 薔薇（ばら）、土竜（もぐら） | 文字から読みが導けない |
| 当て字 | 東雲（しののめ）、五月雨（さみだれ） | 古語・雅語由来の乖離 |
| 意味的当て字 | 小鳥遊（たかなし） | 意味の連鎖を経由した乖離 |
| 完全乖離 | 一（にのまえ） | 文脈なしには復元不可能 |

これらの文字列に共通するのは、**読み手が能動的に解釈に参加しなければならない**という点である。乖離は欠陥ではなく、解釈の余地という豊かさである。

### 暗号理論との接点

乖離した文字列は以下の暗号的性質を自然に備えている。

- **一方向性** — 文字から読みを一意に復元できない
- **衝突耐性** — 同一の読みでも異なる文字列が存在し得る
- **情報密度** — 一つの文字列が複数の解釈を内包する

## ジェネシスブロック：寿限無

J-Genomのジェネシスブロックには落語「寿限無」において子に与えられた名前の文字列を格納する。

```
寿限無、寿限無、五劫の擦り切れ、海砂利水魚の水行末
雲来末、風来末、食う寝るところに住むところ
やぶら小路のぶら小路、パイポパイポ、パイポのシューリンガン
シューリンガンのグーリンダイ、グーリンダイのポンポコピーのポンポコナーの
長久命の長助
```

寿限無は「長命を願う名前をすべて詰め込んだ結果、誰も呼べなくなった」という物語であり、**意味の過積載が機能不全を生む**という逆説を体現している。これはハッシュ関数の衝突耐性の限界——固定長の出力空間に無限の入力を写像する以上、一意性を完全には保てないという数学的必然——と構造的に同一である。

J-Genomはこの逆説を起源として持つことで、価値と機能不全の境界を問い続けるプロジェクトとして自己定義する。

## Divergence Index（乖離指数）

J-Genomは文字列の乖離度を定量化する指標として **Divergence Index（DI）** を定義する。

### 計算要素（案）

| 要素 | 説明 |
|------|------|
| 文字数 / 読み仮名数の比率 | 乖離の基本的な大きさ |
| 読み候補の数 | 解釈の多様性 |
| 読み到達までのステップ数 | 意味的連鎖の深さ |
| 一般認知度の逆数 | 希少性 |

DIが高いほどトークンとしての希少価値が高くなる。

## トークン設計（案）

### JGEM トークン

- **用途** — 乖離文字列のNFT登録・所有権の証明
- **発行** — 登録された文字列のDIに比例
- **希少性** — 同一文字列は一度しか登録できない

### NFT としての文字列

乖離度の高い文字列をNFTとして登録することで以下が可能になる。

- J-Genom上における読み方の「公式解釈」を所有する
- 複数の解釈候補をオークション形式で競う
- 文字列の発見者として記録される

### 登録可能な文字列の範囲

既存の辞書（JMdict等）に収録されている読みは、すでに人類の共有財産である。J-Genomはこれらを登録対象から除外し、最初からパブリックドメインとして扱う。

- **登録不可** — 辞書に収録されている読み（例：薔薇→ばら）
- **登録可能** — 辞書に存在しない読み（例：小鳥遊→たかなし）

この判定はDIの④希少性スコアと同一のデータソース（JMdict）で賄うことができる。

### J-Genom上の所有権の減衰とコモンズへの移行

J-Genom上の所有権は著作権モデルに倣い、時間とともに減衰する設計とする。ここでいう所有権とは現実世界の法的権利ではなく、J-Genomプロトコル内における登録者の優先的地位を指す。

- 登録者は一定期間（例：10年）DIから恩恵を受ける
- 期間経過後はJ-Genom上の所有権がコミュニティに移譲される
- パブリックドメイン化した文字列は「乖離文字列のコモンズ」として誰でも自由に使える

この設計には以下の意図がある：

- **溜め込みより流通を促す** — 永続的な所有より循環を優先する
- **文化資産としての成熟** — 個人の発見がやがて社会の共有財産になる
- **寿限無との整合性** — ジェネシスブロック自体がとっくにパブリックドメインであることと一致する

リソースは使い切れないほど集積するより、必要な人に届く方が健全である。J-Genomはその設計をJ-Genom上の所有権の仕組みに組み込む。

## 多言語拡張の展望

文字と読みの乖離は日本語固有の現象ではない。英語においても綴りと発音の乖離は豊富に存在する。

| 例 | 綴り | 読み |
|----|------|------|
| 地名 | Worcester | /wʊstər/ |
| 地名 | Leicester | /lɛstər/ |
| 普通語 | colonel | /kɜːrnəl/ |
| ジョーク | ghoti | /fɪʃ/（G.B.ショーによる造語） |

J-Genomは将来的に言語モジュールを拡張し、各言語固有の乖離を同一のDIフレームワークで定量化することを目指す。言語ごとにモジュールを分けつつ、共通のDIインターフェースを持つ設計とする。

### リージョンによる空間の分割

J-Genomは `言語コード - リージョンコード` の組み合わせで空間を分割する。トークンの識別子は以下の三要素で構成される。

```
リージョン / 文字列 / 読み
```

例：

| リージョン | 文字列 | 読み | 意味 |
|-----------|--------|------|------|
| JA-JP | 薔薇 | ばら | 日本語・日本 |
| EN-GB | Worcester | /wʊstər/ | 英語・イギリス |
| EN-US | Worcester | /wʊstər/ | 英語・アメリカ |
| JA-KS | 今日 | きょう | 日本語・関西（方言） |

同じ綴り・同じ読みであってもリージョンが異なれば別トークンとなり、発見者もリージョンごとに独立して記録される。これにより方言・地域差が自然に価値の競争軸となる。

リージョンは陣取りゲームの空間でもある。自分のリージョンの言語資産を先に確保する動機が生まれる一方、他リージョンのトークンをあえて取りに行くプレイヤーの存在も想定される。自リージョン外の乖離文字列を発見・登録する行為は、言語的な探検であると同時に、コミュニティ間の競争を生む契機となる。

文字と読みの余白は、人類の言語が共有する普遍的な現象である。
